# Go Settings Boilerplate

This repository provides boilerplate settings for Go projects.
Feel free to use it!

## Prerequisites

This setup is designed for Go development in VS Code with Dev Containers.
It assumes you already have **Docker** installed in your environment.  
If you're new to them, start here:

- **Visual Studio Code**: <https://code.visualstudio.com/download>
- **Dev Containers**: <https://code.visualstudio.com/docs/devcontainers/containers#_installation>

## How to Use

1. Click **"Use this template"** at the top right of this GitHub repository page to create your own copy.
2. Open the repository in **VS Code**, then open the **Command Palette** and select **"Dev Container: Reopen in Container."**
3. That's it - You're all set!

## Tools Included

This boilerplate ships with a few useful tools:

- [**aqua**](https://aquaproj.github.io/): A versatile package manager for managing CLI tools and enforcing consistent versions across your team.
- [**Renovate**](https://docs.renovatebot.com/): Automates dependency updates and version checks.
- [**CSpell**](https://cspell.org/): A spell checker for catching typos early.
- **golangci-lint**, **delve**, and more.
